# AuteNote RaspberryPi
## Bluetooth
/dev/rfcomm0の情報を利用してBluetooth通信を行います．
なので事前にペアリングしておく必要あり．

## 実行方法
'''
python3 main.py
'''
<br>
Python2.Xでは動きません

## -p
を実行時の引数で指定するとカメラモジュールのチェックを実行時に行います．<br>
たぶん

## 写真の運命
写真はスマホ側の要求に応じてraspistillコマンドによって撮影されます．画像は実行時のカレントディレクトリに保存されます．
画像を送信後またはプログラム終了時に削除されます．
